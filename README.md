# 🍔 QuickBite - Responsive Navigation Bar

A modern and responsive navigation bar built using HTML, CSS, and JavaScript as part of Task 4.

The project demonstrates a fully responsive navigation system integrated into a food delivery application interface called **QuickBite**.

---

## 📌 Objective

Create a responsive navigation bar that works seamlessly across desktop, tablet, and mobile devices.

### Requirements Completed

* ✅ Brand Logo
* ✅ Navigation Menu Links
* ✅ Mobile Hamburger Menu
* ✅ JavaScript Toggle Functionality
* ✅ Responsive Design
* ✅ Smooth User Experience

The navigation bar automatically adapts to different screen sizes:

### Desktop

* Full navigation menu visible

### Tablet

* Optimized spacing and layout

### Mobile

* Hamburger menu appears
* Navigation links collapse into a toggle menu
